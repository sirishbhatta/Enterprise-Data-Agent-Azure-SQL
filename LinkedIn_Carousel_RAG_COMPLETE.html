How Memory Works: Cosine Distance

6/9

Converts questions to 768 numbers, measures similarity (0=identical, 1=unrelated)

"Mean salary grouped by department?" 94%
"Show me salaries for each team" 81%
"Top 3 providers by billed amount?" 42%
"What is the weather?" 8%
≥65% = memory hint | ≥95% = skip duplicate

Path Improvements: B, C, D

7/9

B: Follow-Up Rewrite

Problem: "What about just Sales?" (out of context)

Fix: AI expands using history → "What is average salary for Sales?"

C: Memory Hit

Problem: Asked "top providers" before. Now asking "leading providers?"

Fix: 87% match found. Past SQL injected. Badge: "Memory Hit 87%"

D: Model Cascade

Problem: Local AI fails (syntax error)

Fix: Auto-escalate to Gemini → succeeds. Badge: "After 1 retry"
